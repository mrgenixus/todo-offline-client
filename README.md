# Todo Offline Client
## Dependencies
Node 6+ (using 6.11.4)
yarn (npm install -g yarn)

Created using 'create-react-app' [README](CreateReactApp-README.md)

## Setup

1. install with yarn
```
yarn
```

## Running the client
```
yarn start # runs on port 3001
```

## Running the tests (not recommended, or useful)
```
yarn test
```
